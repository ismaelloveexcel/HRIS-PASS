## KEY DESIGN PRINCIPLES

1. **One-Thumb Operation**: Everything accessible with one hand on mobile
2. **3-Click Rule**: Any action completed within 3 clicks/taps
3. **Visual Over Text**: Icons and colors instead of long descriptions
4. **Instant Feedback**: Every action confirmed immediately
5. **Forgiving Design**: Easy to undo/correct mistakes
6. **Progressive Disclosure**: Show advanced features only when needed
